1. Скопировать репозиторий с GitHub на свой компьютер:
   git clone <URL репозитория>на GitHub. 

2. Создать Git репозиторий из локального каталога и отправить его на GitHub: 
   cd <путь к каталогу>
   git init
   git add <имя файла> 
   git commit -m "Описание коммита"
   Создать пустой репозиторий на GitHub.com.
   Скопировать URL-адрес репозитория на GitHub.
   git remote add origin <URL репозитория>
   git push -u origin branch

3. Получить изменения с GitHub на ваш локальный компьютер:
   cd <путь к каталогу>
   git pull 

4. git status -  список измененных, добавленных или удаленных файлов, а также файлов, ожидающих фиксации или отправки на сервер
   git log - показывает список коммитов вместе с их авторами, датами и сообщениями коммитов
   git remote -v - список удаленных репозиториев
